## Default Permission

Default permissions for the pulsar-video plugin (attach/detach surface + decode test clip + audio mute + aspect/orientation control + microphone capture).

#### This default permission set includes the following:

- `allow-attach`
- `allow-detach`
- `allow-play-test`
- `allow-set-audio-muted`
- `allow-set-aspect`
- `allow-set-video-transform`
- `allow-set-orientation`
- `allow-set-status-bar`
- `allow-read-clipboard`
- `allow-screen-refresh-rate`
- `allow-gamepad-battery`
- `allow-get-video-size`
- `allow-get-wallpaper-avatar`
- `allow-mic-start`
- `allow-mic-stop`
